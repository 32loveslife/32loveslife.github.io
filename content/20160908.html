For Windows

首先下载 Shadowsocks Windows客户端 Github Release
或者百度云下载 Ver 3.2 提取密码 2pw5

-->获取账号密码

Shadowsocks可以在VPS搭建，也可以直接购买ss账号.
这里我们以免费的ss账号为例: ishadowsocks 有三个免费实验账号

-->客户端设置

运行下载的 Shadowsocks 程序，在任务栏中找到它(图标是向右飞的纸飞机).
右键---->服务器---->编辑服务器
填写 -->服务器地址(IP) -->端口 -->密码 -->并选择加密方式。
备注可以随便写。代理端口一般设置为1080即可。

-->启用系统代理

右键ss 确认已经勾选 <启用系统代理>

-->完成

打开浏览器
输入 https://www.google.com/ncr
或者 https://www.youtube.com/
Enjoy yourself.

-->What if fails ?

1:更换ss账号
2:检查DNS
